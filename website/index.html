Argonbox

A really basic and lightweight lightbox plugin for jQuery. Easy to use, modify and expand. Developed in a JavaScript course at Blekinge Institute of Technology.

Download

GitHub

Demo

Usage

index.html

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>Template</title>
  <link href="argonbox/css/argonbox.css" rel="stylesheet">
</head>
<body>
  <h1>Hello world!</h1>
  <div class="argonbox">
    <a href="img/world.jpg" title="Hello world">
      <img src="img/world.jpg" alt="World" />
    </a>
  </div>
  <script src="http://code.jquery.com/jquery-latest.js"></script>
  <script src="argonbox/js/argonbox.js"></script>
  <script src="main.js"></script>
</body>
</html>

main.js

$(document).ready(function() {
  'use strict'; // Prevents certain (unsafe) actions from being taken and throws more exceptions.
  // Determining when the function is called
  $('.argonbox a').click(function() {
    $(this).argonbox({'duration':'fast'});
    return false; // Prevent the default behavior of the HTML link.
  });
});
Tip! Argonbox looks good on tablets and smartphones.
